Python 开发环境管理

本文主要记录了 Python 开发环境管理的方法，包括 Miniconda 管理 Python 开发环境、Pyenv 和 Pipenv 管理 Python 开发环境等。

Python 开发环境管理分为两部分，一个是 Python 版本管理，一个是虚拟环境管理。

虚拟环境管理存在两种方式，第一种是虚拟环境存放于项目文件夹，第二种情况是虚拟环境存放于系统指定文件夹，这时候不同的项目之间可以复用同一个虚拟环境。
