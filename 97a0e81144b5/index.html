Hexo Blog 中支持 Emoji

问题描述

今天在新建博客的时候，发现 Hexo 不支持 Markdown Emoji，查阅资料发现原因是因为 Hexo 默认的渲染器是 hexo-renderer-marked，这个默认的渲染器不支持 Emoji 功能。如果希望 Hexo 支持 Emoji，就需要换一个渲染器。

这里使用知乎博主推荐的 hexo-renderer-markdown-it 渲染器，支持扩展，采用的是 markdown-it 的内核来解析 Markdown 的文本。
