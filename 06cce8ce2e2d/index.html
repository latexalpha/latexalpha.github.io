Obsidian 基本介绍

软件使用哲学

使用 Obsidian 记录的原则在于：一切的笔记文件都存储于自己的设备之上，以 Markdown 文件的形式。

所以需要熟练掌握 Markdown 笔记的书写规范，以及形成自己的笔记整理的工作流。

软件功能

  • Settings 里的 Auto convert HTML 可以在粘贴的时候将链接自动转换为 Markdown 格式的链接。

软件快捷键

由于是在 Obsidian 中，所以快捷键不区分大小写，这里为了可读性，直接用大写字母代替。

快捷键 功能
Ctrl + E 切换阅读模式与编辑模型
Ctrl + L 新建代办事项
